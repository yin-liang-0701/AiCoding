### 3. 修改建议与分析

#### ✅ usecase.py
```python
# 新增用例类（在文件末尾添加）
class ExportMemoAsJsonUseCase:
    def __init__(self, user_id):
        self.user_id = user_id
        
    def execute(self):
        # 获取当前用户的Memo列表（复用ListMemosUseCase逻辑）
        memo_repo = SqliteMemoRepository()
        memos = memo_repo.find_all_by_user(self.user_id)  # 需要repository支持
        
        # 转换为字典结构（实体层保持纯净，转换放在用例层）
        return [{
            "id": memo.id,
            "content": memo.content,
            "created_at": memo.created_at.isoformat(),
            "tags": [tag.name for tag in memo.tags]
        } for memo in memos]
```

#### ✅ controller.py
```python
# 新增命令处理器（在函数列表末尾添加）
def handle_export_json():
    user_id = get_current_user_id()
    if not user_id:
        view.show_login_required()
        return
        
    export_usecase = ExportMemoAsJsonUseCase(user_id)
    memo_data = export_usecase.execute()
    view.show_exported_json(memo_data)

# 在main()函数中添加命令路由（在if-elif链中添加）
elif command == "export-json":
    handle_export_json()
```

#### ✅ view.py
```python
# 新增JSON输出方法（在文件末尾添加）
import json  # 在文件顶部添加导入

def show_exported_json(memo_data):
    """输出JSON格式的Memo数据"""
    print(json.dumps(memo_data, ensure_ascii=False, indent=2))
```

#### ⚠️ repository.py（可能需要）
```python
# 如果现有find_all不支持用户过滤，在SqliteMemoRepository中添加：
def find_all_by_user(self, user_id):
    conn = self._get_conn()
    cursor = conn.cursor()
    cursor.execute("SELECT * FROM memos WHERE user_id = ?", (user_id,))
    rows = cursor.fetchall()
    return [self._row_to_memo(row) for row in rows]
```

### 4. 关键实现细节
1. **权限控制**：在controller层验证用户登录状态（复用现有`get_current_user_id`）
2. **数据转换**：
   - 在usecase层将实体转换为字典（避免污染实体层）
   - 时间格式使用ISO标准（`isoformat()`）
3. **JSON输出**：
   - 使用`ensure_ascii=False`支持中文
   - `indent=2`美化输出
4. **复用现有逻辑**：
   - 复用`ListMemosUseCase`的查询逻辑
   - 复用`_row_to_memo`等数据转换方法
