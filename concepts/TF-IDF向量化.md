# TF-IDF
是一种统计方法，用以评估字词对一个文件集或者语料库中的其中一份文件的重要程度。是信息检索与文本挖掘常用的加权技术。又叫做 **词频-逆文档频率**

## 计算公式

tf = n / N
- n: 某个文档中出现的次数
- N： 所有文档中出现的次数

idf = lg(D / d)
- D: 总的文档数
- d: 词语所在文档数

tf-idf = tf * idf  
会随词语出现的频率增大而变大，也会随这个词在不同文档里出现的次数的增多而下降。  
达到突出重要词语，抑制次要词语的作用
